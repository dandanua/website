hyp bee put

Set the value of an entry of the given hyperbee URL.

hyp bee put {url} [value]

Examples:

hyp bee put hyper://1234..af/foo "hello"
hyp bee put hyper://1234..af/foo/bar 12345
cat data.json | hyp bee put hyper://1234..af/data
